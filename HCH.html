Hand Cannon Havoc

Project Overview

Description

Developed a 2D shooting platformer using Unity. Implemented core game systems such as basic player movement, special movement abilities, weapon systems, and AI enemies through custom C# scripts.

Skills Learned

  • Unity for 2D game development.
  • C# Scripting for game logic and physics.
  • designing and implementing enemies with AI behaviors.
  • using GitHub to collaborate with team members.
