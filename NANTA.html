NANTA | Not Another Note Taking App

Project Overview

Description

Developed a note-taking application using a MERN stack (MongoDB, Express.js, React, and Node.js). The app supports standard rich text editing options and note saving. The user interface was designed using React, ShadCN, and TailwindCSS. The app also has a mobile implementation through the use of Dart and Flutter..

Skills Learned

  • Front-end development using React, ShadCN, and Tailwind.
  • Mobile development with Flutter and Dart.
  • Docker containerization for consistent testing environments.
  • GIT version control and use of branches for easy team collaboration.
